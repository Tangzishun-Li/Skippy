# 逃课记录软件

一个使用Electron框架开发的逃课记录软件，用于记录课程的出勤情况和学习问题，方便回顾需要复习的内容。

## 功能特点

- ✅ 添加课程：支持设置课程名称、上课时间、星期几和重复周数
- ✅ 状态管理：
  - 未来课程：灰色
  - 已上且无问题：绿色
  - 未上/旷课：红色
  - 已上但有问题：黄色
- ✅ 问题记录：点击黄色课程可记录学习问题
- ✅ 数据持久化：使用本地存储，数据不丢失
- ✅ 响应式设计：适配不同屏幕尺寸

## 如何打开和使用

### 前提条件

确保您的电脑已安装Node.js环境。如果未安装，请先下载并安装：
- [Node.js官方下载](https://nodejs.org/zh-cn/download/)

### 安装依赖

1. 打开命令提示符（CMD）或PowerShell
2. 导航到项目目录：
   ```bash
   cd d:\code_space\record_every_skip
   ```
3. 安装项目依赖：
   ```bash
   npm install
   ```

### 运行软件

在项目目录下执行以下命令：

```bash
npm run electron
```

### 使用教程

1. **添加课程**：
   - 在"添加课程"表单中填写课程信息
   - 选择星期几、开始和结束时间
   - 设置重复周数（如14周）
   - 点击"添加课程"按钮

2. **管理课程状态**：
   - 点击每周的课程项
   - 在弹出的菜单中选择对应状态
   - 状态会立即更新并保存

3. **记录问题**：
   - 将课程状态设置为"已上（有问题）"
   - 点击黄色的课程项
   - 在弹出的表单中输入问题描述
   - 点击"保存问题"按钮

4. **查看记录**：
   - 课程列表会显示所有课程及其状态
   - 有问题的课程会显示"有问题"标签
   - 数据会自动保存，下次打开时会恢复

## 常见问题

1. **无法运行软件**：
   - 检查Node.js是否已正确安装
   - 确保所有依赖已成功安装
   - 尝试以管理员身份运行命令提示符

2. **数据丢失**：
   - 软件使用localStorage存储数据，通常不会丢失
   - 若浏览器清除了本地存储，数据可能会丢失
   - 建议定期备份重要数据

3. **界面显示异常**：
   - 尝试调整窗口大小
   - 检查是否有CSS样式冲突
