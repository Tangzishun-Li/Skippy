# Skippy

一个使用Electron框架开发的逃课记录软件，用于记录课程的出勤情况和学习问题，方便回顾需要复习的内容。

## 功能特点

- ✅ 添加课程：支持设置课程名称、上课时间、日期、频率和重复次数
- ✅ 状态管理：
  - 无标签：默认状态
  - 已上且无问题：绿色
  - 未上/旷课：红色
  - 已上但有问题：黄色
  - 未来课程DDL：蓝色
- ✅ 问题记录：点击黄色课程可记录学习问题，悬停显示问题详情
- ✅ 日历视图：直观显示课程安排和状态
- ✅ 数据持久化：使用本地存储，数据不丢失
- ✅ 响应式设计：适配不同屏幕尺寸
- ✅ 当前周标记：自动标记当前周的课程
- ✅ 清新界面：现代化的UI设计，使用柔和的配色方案

## 如何打开和使用

### 前提条件

确保您的电脑已安装Node.js环境。如果未安装，请先下载并安装：
- [Node.js官方下载](https://nodejs.org/zh-cn/download/)

### 安装依赖

1. 打开命令提示符（CMD）或PowerShell
2. 导航到项目目录：
   ```bash
   cd d:\code_space\record_every_skip
   ```
3. 安装项目依赖：
   ```bash
   npm install
   ```

### 运行软件

在项目目录下执行以下命令：

```bash
npm run electron
```

### 使用教程

1. **添加课程**：
   - 在"添加课程"表单中填写课程信息
   - 选择开始和结束时间
   - 选择上课日期
   - 选择频率（每周、每两周、每个月或每天）
   - 设置重复次数
   - 点击"添加课程"按钮

2. **管理课程状态**：
   - 点击每周的课程项（圆形进度点）
   - 在弹出的菜单中选择对应状态
   - 状态会立即更新并保存
   - 未来课程可以设置为"无标签"或"DDL"
   - 过去课程可以设置为"已上（无问题）"、"未上/旷课"或"已上（有问题）"

3. **记录问题**：
   - 将课程状态设置为"已上（有问题）"
   - 点击黄色的课程项
   - 在弹出的表单中输入问题描述
   - 点击"保存问题"按钮
   - 悬停在黄色课程项上可以查看问题详情

4. **查看日历**：
   - 在"日历视图"中查看课程安排
   - 当天日期会有特别的蓝色标记
   - 不同状态的课程会以不同颜色显示
   - 可以通过导航按钮切换月份

5. **查看课程列表**：
   - 在"课程列表"中查看所有课程及其状态
   - 有问题的课程会显示"有问题"标签
   - 当前周的课程会有黑色边框标记
   - 数据会自动保存，下次打开时会恢复

## 技术实现

- **前端**：HTML5 + CSS3 + JavaScript
- **框架**：Electron
- **构建工具**：Vite
- **数据存储**：localStorage

## 常见问题

1. **无法运行软件**：
   - 检查Node.js是否已正确安装
   - 确保所有依赖已成功安装
   - 尝试以管理员身份运行命令提示符

2. **数据丢失**：
   - 软件使用localStorage存储数据，通常不会丢失
   - 若浏览器清除了本地存储，数据可能会丢失
   - 建议定期备份重要数据

3. **界面显示异常**：
   - 尝试调整窗口大小
   - 检查是否有CSS样式冲突
