# Electron逃课记录软件开发计划

## 项目初始化
1. 使用 `npm create vite@latest` 初始化一个基本项目
2. 安装Electron相关依赖
3. 配置项目结构和构建脚本

## 核心功能实现

### 1. 课程管理模块
- 添加课程功能：输入课程时间（如每周9-11点）、课程名称、重复周数
- 课程列表展示
- 课程编辑和删除功能

### 2. 状态管理模块
- 课程状态定义：
  - 未来课程：灰色
  - 已上且无问题：绿色
  - 未上/旷课：红色
  - 已上但有问题：黄色
- 状态手动切换功能
- 鼠标交互黄色课程时的问题记录功能

### 3. 数据存储模块
- 使用本地存储（如localStorage或sqlite）保存课程数据
- 数据结构设计：包含课程信息、状态信息、问题记录等

### 4. 界面设计模块
- 主界面：课程列表视图
- 添加/编辑课程表单
- 问题记录弹窗
- 响应式设计，确保在不同屏幕尺寸下正常显示

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript
- 框架：Electron
- 构建工具：Vite
- 样式：原生CSS或轻量级CSS框架
- 数据存储：localStorage（简单实现）或sqlite（更持久化）

## 实现步骤
1. 搭建基础项目结构
2. 实现主窗口和基本布局
3. 开发课程添加功能
4. 实现课程状态管理和显示
5. 开发问题记录功能
6. 实现数据存储和加载
7. 测试所有功能并优化用户体验
8. 构建最终应用
